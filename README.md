**BeautyShop** — це веб-сайт для продажу косметики, який демонструє навички роботи з HTML, CSS та Git/GitHub.  
Проект включає декілька сторінок з різним контентом та функціоналом.

Сторінки сайту
- **Головна (index.html)** – привітання, опис магазину, популярні товари.  
- **Про нас (about.html)** – інформація про магазин та команду.  
- **Статті (articles.html)** – корисні статті про догляд за шкірою та косметику.  
- **Контакти (contacts.html)** – контактна інформація та форма для зв’язку.  
- **Галерея (gallery.html)** – фото продуктів із підписами.  
- **Форма (form.html)** – складна форма для замовлень або запитів.  
- **Якорі (anchors.html)** – демонстрація роботи з якорями на сторінці.

Особливості сторінки:
- Адаптивний дизайн через CSS.  
- Використані різні типи селекторів: тег, клас, id, атрибут, вкладені та послідовні селектори.  
- Галерея з 3-ма колонками.  
- Центральне меню на всіх сторінках.  
- Валідація HTML та CSS через онлайн-валідатори.
